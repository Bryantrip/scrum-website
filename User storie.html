Een user verwacht een website met informatie over games omdat hij of zij iets wil weten. De user komt terecht op de homepagina en ziet het thema ( gaming ) en dat hij of zij deze website waarschijnlijk kan gebruiken voor de benodigde informatie. De user klikt op 1 van de knoppen van de navbar en ontdekt dat de pagina's gesorteerd zijn op maanden van 2019. De user vind de informatie en verlaat de website.
